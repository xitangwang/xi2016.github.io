改变3：健走

2017-07-05 | 2017-07-05 |
|

改 变

每天健走2～5分钟。如果你已经养成健走的好习惯，可以增加到10分钟或者更久。

理 由

年纪大了，身体重要。健走是我见过最简单易行的保健方法了。如果你喜欢其他运动（比如健身，游泳），也可以把健走作为补充运动。相比那些做1休6、做3休4的运动，健走可以每天都进行。

另外，我在健走时，常常会有些奇思妙想呢。

步 骤

  1. 尽量每天选择一个固定的时间健走（比如吃完晚饭）；

  2. 每天只健走2-5分钟。如果你已经养成每天健走的好习惯，可以增加到10分钟或更久；

  3. 不需要啥乱七八糟的准备，穿上鞋就走，以放松的方式，缓慢地走；

  4. 如果你感觉良好，可以每天延长2-3分钟。

以我的经验，带上耳机或者小音箱健走，会增加不少的乐趣。另外一个让你爱上健走的好方法是和朋友、家人、恋人一起走。书中介绍的很多改变，其实你都可以找个伴一起完成。同伴的相互监督和支持，会让大家更容易坚持。

调整

根据实际情况，改为每天下午跑完步后在运动场上走两圈。
